# Pokémon Guesser App

An interactive Pokémon search and discovery application built with vanilla JavaScript, HTML, and CSS. The app integrates with the PokéAPI to provide detailed Pokémon information, including base stats, descriptions, abilities, evolution chains, and image galleries.

## Live Features

### Pokémon Search

* Search for Pokémon by name
* Autocomplete suggestions while typing
* Fast and responsive search experience

### Detailed Pokémon Information

* Pokémon name and ID
* Height and weight measurements
* Pokémon types with colour-coded badges
* Official artwork and sprite gallery

### Stats & Visualisation

* Animated stat counters
* Interactive radar chart built with Chart.js
* Displays:

  * HP
  * Attack
  * Defense
  * Special Attack
  * Special Defense
  * Speed

### Pokédex Information

* Official Pokédex descriptions
* Pokémon abilities
* Evolution chain display
* Evolution requirements and methods

### User Experience

* Dynamic type-based colour themes
* Smooth animations and transitions
* Responsive mobile-friendly layout
* Loading indicators and visual feedback

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Chart.js
* PokéAPI

## Installation

Clone the repository:

```bash
git clone https://github.com/lorelei02/pokemon-stats-app.git
```

Navigate to the project folder:

```bash
cd pokemon-stats-app
```

Open `index.html` in your browser.
